Portfolio Performance

Berechne die Performance eines Portfolios anhand aller Transaktionen und über alle Konten hinweg inklusive internen Zinsfuß.

  • Windows 32bit / 64bit [?]Portfolio Performance ist in Java programmiert. Darum kommt es darauf an, ob die Java Virtual Machine (JVM) in der 32bit oder 64bit Version installiert ist. Das kann man mit dem Befehl java -version auf der Kommandozeile feststellen.
  • Mac OS X (benötigt Java [?]Portfolio Performance ist in Java programmiert und benötigt mindestens Java 7. Allerdings wird die Java Laufzeitumgung nur bis einschließlich Version 6 direkt von Apple ausgeliefert. Sie kann von dem nebenstehenden Link heruntergeladen werden.)
  • Linux 32bit / 64bit

Aktuelle Version 0.14.0 (via Online Aktualisierung & Download) - was ist neu?

Features

  • Historische Performance auf Basis aller Transaktionen und Dividenden
  • Performance-Berechnung nach der True-Time Weighted Rate of Return Methode
  • Berechnung des internen Zinsfußes (Internal Rate of Return)
  • Aggregation mehrerer Konten (z.B. Tagesgeldkonten) und mehrerer Portfolios
  • Vergleich mit den deutschen Verbraucherpreisen (Inflationsrate)
  • Aktualisierung der Kurse von Yahoo Finance und beliebigen HTML Seiten
  • Offenes Dateiformat: alle Daten sind als XML gespeichert oder können als CSV exportiert werden

Worum geht es nicht?

  • Kein Online Banking und keine Überweisungen
  • Ungeeignet für Girokonten - viel zu viele Bewegungen
  • Keine Kategorisierung von Einnahmen/Ausgaben
